# Versions

## v0.0.0 -> v0.0.1

- Init repo

## v0.0.1 -> v0.0.2

- Создан стартовый skeleton проекта BBSim.
- Добавлен pipeline эпох с первыми stage-ами: personal seed, inflation, recombination preview.
- Добавлен NumPy backend для deterministic seed field и базовых операций с полями.
- Добавлен минимальный PySide6/PyQtGraph UI.
- Добавлены базовые тесты детерминизма seed-а, формулы расширения и порядка pipeline.

## v0.0.2 -> v0.0.3

- Синхронизирована версия проекта до 0.0.3 в package metadata, `__version__` и заголовке окна.
- Улучшена генерация личного seed field: вместо сырого белого шума используется сглаженное Fourier-shaped поле с крупномасштабной структурой и слабым fine layer.
- Переработаны метрики seed-а: крупномасштабный рисунок, мелкая зернистость, потенциал пустот и риск раннего коллапса считаются устойчивее.
- Добавлена человекочитаемая сигнатура зерна в checkpoint report.
- Добавлена космическая цветовая карта для центрального поля и нижний timeline текущих checkpoint-ов.
- Начальный масштаб `a` сдвинут глубже в раннюю эпоху, чтобы после инфляции preview-рекомбинация не определялась как поздняя dark-energy эпоха.
